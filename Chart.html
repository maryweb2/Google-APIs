Google Chart

Esta API permite la creación e implementación de gráficos para diferentes estilos. Google Charts combina las dos interfaces de programación de aplicaciones(API), que son Google Chart API(crea visualizaciones estáticas de datos) y Google Visualization API.


Visualización

  • Líneas de tiempo.

  • Mapas de calor.

  • Mapas de árboles

  • Nubes de palabras o términos

  • Filtros para otras visualizaciones

  • Google maps interactivo

Ventajas

  • Tiene gran variedad de gráficas.

  • Sirve para representar datos.

  • Brinda la opción de personalizar las gráficas.

  • Brinda la posibilidad de interactuar con el gráfico.

Desventajas

  • Funciona con javascript con los problemas de seguridad que este estándar puede conllevar.

  • Clausulas del contrato o licencia de Google: Para ello google se reserva el derecho, entonces de este modo sugiere que agreguen publicidad en el contenido a través de las API.

  • Limitaciones de las API: Google limita el número de solicitudes de API de los usuarios.

  • Depende de APIs externos:Cualquier error que se cometa al desarrollarlo no puede ser controlado.
