Google Fonts

Es un directorio de uso libre por un programa de interfaces para fuentes de la web. Están diseñadas para proporcionar una plataforma gratuita para descubrir nuevas fuentes para los usuarios.


Funciones

  • Algunas fuentes funcionan especialmente para títulos y otras para párrafos.

  • Se utilizan para darle una mejor apariencia a la página ya que incluye tipos de letras en mayúsculas y cursivas.

  • Las google fonts son adaptables para la página que se requiera, utilizando el estilo que el usuario desea.

Ventajas

  • Es un servicio gratuito: No importa cuantas fuentes quieras usar ni en cuantos sitios web. Lo bueno de esto es que no necesitamos de una licencia.

  • Más de 850 fuentes para elegir: Es un catálogo realmente extenso para seleccionar e implementarlo en nuestro proyecto.

  • La aplicación web funciona muy bien: Google cuenta con un catálogo bien organizado y categorizado para mejor visualización del usuario.

  • Hace tu web más rápida: Las fuentes se alojan en los servidores de Google para que los archivos que están comprimidos se puedan colocar rápidamente en el navegador.

  • Es fácil de implementar: Solo “copia y pega” en la página HTML.
